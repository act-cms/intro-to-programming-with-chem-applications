Lesson 01: Introduction to Transitioning from Excel to Python for Chemical Lab Data Analysis Pre Lab Activity
=======================

The purpose of this lesson is to introduce Python and the basics of a Jupyter  Notebook.

## Lesson Information
### Lesson Learning Outcomes
At the end of this lesson, students will be able to...
1. Describe the Python programming language and its uses.
2. Learn the basics of the Jupyter notebook.
